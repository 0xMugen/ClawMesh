# Protocol Spec (`v0.1`)

This folder defines the human-readable protocol reference for ClawMesh.

## Layout

- `protocol-v0.1.md`: normative envelope + message semantics.
- `examples/`: complete message examples for `announce`, `request_match`, `schedule`, and `offer`.
  - `announce.json`
  - `request_match.json`
  - `schedule.json`
  - `offer.json`

## Versioning

- Current bootstrap version: `0.1`.
- Backward-incompatible changes require a new major/minor schema directory.
